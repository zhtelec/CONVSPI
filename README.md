# CONVSPI
SPI 信号電圧変換基板<br/>
SCLK, MOSI, MISO, CSX, RESETX, INT1/2 の ラインの信号電圧を変換<br/>
マスター側の 1.2V から 5V までの信号電圧をスレーブ側の 1.2V から 5V までの信号電圧に変換<br/>
CSX はインアクティブ時はマスター側の MISO をハイインピーダンス<br/>
